# Contributing to Chess Coach ♟️

First off, thank you for considering contributing to Chess Coach! It's people like you that make it such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce it.
- Include information about your browser and OS.

### Suggesting Enhancements
- Open a GitHub issue with a clear title and description.
- Explain why the enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Style Guide
- Follow the existing code style (standard React/Vite/ESLint).
- Use clear, descriptive variable and function names.
- Comment complex logic where necessary.
